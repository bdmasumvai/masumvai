# Masum Vai - Personal Bio Website

A professional, dark-themed personal portfolio website designed for Masum Vai.
Built with clean HTML, CSS, and Vanilla JavaScript.

## Structure
- **Boot Screen**: Minimal terminal-style intro.
- **Profile**: Clean typography and code-style biography.
- **Socials**: Grid layout for professional links.
- **Music**: Minimal floating audio player (Quran Recitation).

## Deployment
1. Upload all files to your host or GitHub Repository.
2. Ensure `assets/img/profile.jpg` exists.
3. Ensure `assets/music/quran.mp3` exists.
4. Enable GitHub Pages (if using GitHub).
